# Overview

<p>ZXing ou Zebra Crossing como tambem é chamada, é uma biblioteca open source feita em Java, para processamento de imagem em formato barcode. Ela tambem foi portada para outras linguagens como PHP, Javascript, C++, Delphi. Neste repositorio/projeto, mostro de maneira basica como implementar um QRCode,para 9 funcionalidades. A biblioteca ZXing, tem mais funções e muito mais customizações, portanto você pode acessar o github do ZXing para melhores informações. O projeto é focado mais no resultado final, então ele é uma API simples, com endpoints get que irão gerar arquivos PNG com QRcodes e teve como finalidade gerar uma base de conhecimento e entender melhor sobre a biblioteca. Ele tambem é destinado a ajudar qualquer pessoal interessada em saber mais das possibilidades da utilização da biblioteca, em especial do QrCode.</p>
